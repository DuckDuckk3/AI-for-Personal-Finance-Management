AI QUẢN LÝ TÀI CHÍNH CÁ NHÂN

📌 Giới thiệu
  - AI này giúp người dùng theo dõi thu nhập, chi tiêu, lập kế hoạch tài chính và đưa ra khuyến nghị tối ưu hóa chi tiêu.

🚀 Tính năng chính
  - Phân loại giao dịch tự động dựa trên mô hình Machine Learning.
  - Dự đoán số dư tài khoản dựa trên lịch sử giao dịch.
  - Lập ngân sách cá nhân để kiểm soát chi tiêu.
  - Chatbot tài chính hỗ trợ người dùng qua giao diện hội thoại.

📂 Cấu trúc dự án
  📁 financial-ai
  │── 📂 data             # Dữ liệu giao dịch (CSV, JSON)
  │── 📂 models           # Mô hình Machine Learning
  │── 📂 scripts          # Code xử lý dữ liệu
  │── 📂 chatbot          # Chatbot tài chính
  │── 📂 frontend         # Giao diện web/app
  │── 📜 README.md        # Tài liệu dự án
  │── 📜 requirements.txt # Danh sách thư viện cần cài đặt

🔧 Cài đặt & Sử dụng

1️⃣ Clone Repository
  git clone https://github.com/your-username/financial-ai.git
  cd financial-ai

2️⃣ Cài đặt thư viện
  pip install -r requirements.txt

3️⃣ Chạy thử hệ thống
  python scripts/main.py

🛠 Công nghệ sử dụng
  - Python: Ngôn ngữ lập trình chính.
  - Pandas, NumPy: Xử lý dữ liệu.
  - Scikit-learn, TensorFlow: Phát triển mô hình Machine Learning.
  - FastAPI: API backend.
  - Streamlit: Giao diện hiển thị dữ liệu.
  - Telegram Bot: Chatbot tài chính.
